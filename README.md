# CzZ_EasyLan

一个简单的局域网通讯模块，Android 原生实现，零第三方依赖。

## 这是什么

这不是一个完整的 App，而是一个可以放到任何项目里使用的通讯模块。

UDP 广播发现、Socket 连接、心跳保持、数据收发 —— 全部自己实现。

## 能做什么

- 局域网内自动发现设备
- 建立点对点连接
- 稳定保持长连接
- 与 iOS / PC 数据互通（协议通用）

## 技术栈

- Android 原生 API
- 无第三方框架
- 自定义通讯协议
- 完整数据模型

## 谁可以用

- 想学局域网通讯的开发者
- 需要在项目里集成通讯功能的人
- 想移植到其他平台的人
